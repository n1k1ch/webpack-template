# Template for building Web Application using [Webpack](https://webpack.github.io/docs/tutorials/getting-started/)
## "Features"

- bundling SCSS and JS using webpack
- processing SCSS
- processing Jade


## Prerequisites

```dos
npm install webpack -g
```

## Initial build

```dos
npm update
npm run build
```

## Developer mode

```dos
npm run watch
```

## Run
 Open `dist/index.html` in browser
