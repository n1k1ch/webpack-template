# Template for building Web Application using [Webpack](https://webpack.github.io/docs/tutorials/getting-started/)
## "Features"

- bundling CSS and JS using webpack
- processing SCSS
- processing Jade


## Prerequisites

```bat
npm install webpack -g
```

## Initial build

```bat
npm update
npm run build
```

## Developer mode

```bat
npm run watch
```

## Run
 Open `dist/index.html` in browser
